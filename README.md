# Interview Challenge - Stellar Blockchain

A coding challenge for working with Stellar blockchain data.

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/YOUR_REPO)

### Quick Deploy

1. Push this repository to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your GitHub repository
4. Click "Deploy"

That's it! Vercel will automatically detect this as a static site.

## Local Development

```bash
# Install serve (optional, for local preview)
npm install

# Run locally
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000)
